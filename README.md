## OSmit

Implement a simple operating system.

### Execution
```bash
$ cd boot
$ make
# run on qemu-system-i386
$ make qemu
```

### Compilation
Not so serious to build own cross-compilation toolchain(binutils, gcc), use flags in gcc, ld instead.
